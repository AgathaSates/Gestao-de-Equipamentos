## Introdução

O projeto **Gestão de Equipamentos** é uma aplicação que simula o gerenciamento de equipamentos, com funcionalidades como cadastro, listagem, edição e exclusão de registros.

## Tecnologias usadas

[![Tecnologias](https://skillicons.dev/icons?i=git,github,cs,dotnet,visualstudio)](https://skillicons.dev)

## Funcionalidades

- **Cadastro:** Registre novos equipamentos e chamados com dados como nome, número de série, data de aquisição, entre outros.
- **Listagem Detalhada:** Visualize todos os equipamentos cadastrados.
- **Edição:** Modifique as informações de equipamentos já cadastrados.
- **Exclusão:** Remova itens da lista de forma prática.
- **Interface Interativa no Console:** Utilização de menus e prompts para facilitar o uso .

## Como Usar

### Requisitos

- .NET SDK (.NET 8.0 ou superior recomendado)

### 1. Clone o repositório.
 
```
git clone https://github.com/AgathaSates/Gestao-de-Equipamentos.git
```
### 2. Abra o terminal ou o prompt de comando e navegue até a pasta raiz

```
cd cd Gestao-de-Equipamentos
```

### 3. Utilize o comando abaixo para restaurar as dependências do projeto.

```
dotnet restore
```

### 4. Navegue até a pasta do projeto

```
cd cd Gestao-de-Equipamentos.ConsoleApp
```

### 5. Execute o projeto

```
dotnet run
```
